# Taskator - Daily Goal Tracker

Taskator is a modern daily goal tracking application built with React, Vite, TypeScript, and Supabase.

## Features

- Daily goal tracking with easy completion toggle
- Drag and drop reordering of tasks
- Statistical insights and activity history
- Secure authentication with optional 2FA
- Responsive design for mobile and desktop

## Tech Stack

- **Frontend:** React, Tailwind CSS, shadcn/ui
- **Backend:** Supabase (Auth, Database, RLS)
- **Deployment:** Vercel (Paris region)
